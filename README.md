# Rapport de bugs SchoolMore

Utilisez ces procédures afin de signaler un dysfonctionnement sur l'une de nos applications SchoolMore *(scm-app, scm-serv, scm-mobile, etc.)* et nous permettre de régler le problème dans les plus brefs délais.
Vous pouvez également utiliser ces liens pour demander une nouvelle fonctionnalité ou l'amélioration d'un outil existant.

-------

## 🐞 Rapport de bugs


### Via le site schoolmore.fr :

1. Rendez-vous sur la page **Contact** *(située dans le pied de page, tout en bas)*.
2. Cliquez sur le lien **Signalement de bugs**.
3. Créez un compte ou connectez-vous à votre compte GitHub*.
4. Remplissez le formulaire en étant le plus précis possible.
5. Envoyez votre demande.

### Via GitHub :

1. Recherchez le dépôt (repository) nommé **scm-issues**.
2. Dans l'onglet **Issues**, cliquez sur "New Issue" et sélectionnez le modèle **Bug report**.
3. Complétez les informations demandées avec précision.
4. Validez l'envoi de votre demande.

-------

## ✨ Nouvelles fonctionnalités

### Via le site schoolmore.fr :

1. Rendez-vous sur la page **Contact**.
2. Cliquez sur le lien **Demande de fonctionnalités**.
3. Créez un compte ou connectez-vous à votre compte GitHub.
4. Décrivez votre besoin en étant le plus précis possible.
5. Envoyez votre demande.

### Via GitHub :


1. Recherchez le dépôt **scm-issues**.
2. Dans l'onglet **Issues**, sélectionnez le modèle **Feature request**.
3. Détaillez votre proposition d'amélioration.
4. Validez l'envoi de votre demande.

------
*Note : GitHub est un service externe à SchoolMore. Aucune information personnelle transmise à SchoolMore ne sera envoyée à GitHub. SchoolMore décline toute responsabilité en cas de problème survenu sur la plateforme GitHub.
